### 全局函数
```js
function f1() {}
function f2() {}
```
```
缺点：
1. 容易造成全局环境的污染(命名污染)
2. 模块之间的关系复杂
```

### namespace
```js
  var myModule = {
    data: 'value',
    foo() {
      console.log('foo', this.data)
    },
    bar() {
      // xxx
    }
  }
  myModule.foo()
  myModule.bar()
  var myModule2 = {}
  //  ...
```

```txt
缺点
1. 容易修改到别人的代码(通过myModule.data = xxx之间修改了代码)
2. myModule会打印出所有的属性和方法(我可能只需要其中的某个方法)
```
